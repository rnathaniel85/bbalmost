Bad Bank MERN App

About 
"Bad Bank is a Three tier banking App that has Front End, Back End and Database capabilities. 

Components 
- React
- Node.js
- MongoDB
- Bootstrap
- Firebase

Install
1. Clone repo
  - gh repo clone rnathaniel85/badbankfinal
2. Install Packages
  - npm install
3. Run Node
  - node index.js
